# Stat Mod 

## Each Command will set the game mode to offline to promote offline play
### Commands
* F2: Lots of vigor ( 99999999 )
* F3: Super health, stamina, magic, and item discovery
* F4: Max level weapons in inventory
* F5: Toggle infinite consumable use for items that are considered usable in your inventory

#### Required: 
* Launch game from [EACBEGONE](https://www.nexusmods.com/lordsofthefallen2023/mods/54?tab=description)
* Latest ZDev from [UE4SS](https://github.com/UE4SS-RE/RE-UE4SS/releases)

#### Install
* Install ue4ss into your games Win64 dir. Example: D:\SteamLibrary\steamapps\common\Lords of the Fallen\LOTF2\Binaries\Win64
* unzip StatMod into Mods Directory.  Example: D:\SteamLibrary\steamapps\common\Lords of the Fallen\LOTF2\Binaries\Win64\ue4ss\Mods
* Enable in mods/mods.txt.  
    * Add the below line
        * **StatMod: 1** 

##### Notes
* UE4SS will only launch if you launch game from EACBEGone Mod
* This mod is intended for OFFLINE use only.  
* Only tested to work with ue4ss experimental-latest **zDEV-UE4SS_v3.0.1-946-g265115c0.zip**, but may work with other versions.
